# Rules

To contribbute to the codebase, our syntax rules have to be followed:

- snake_case for functions  
    ✔️  `int myFunc(void);`  
    ❌ `int my_func(void);`  

- PascalCase for custom types (typedef)
    ✔️  `int myFunc(void);`    
    ❌ `int my_func(void);`    

- if a function does not accept parameters, explicit it  
    ✔️ `int myFunc(void);`  
    ❌ `int myFunc();`
